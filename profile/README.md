# CipherBridge

Cloud-native MT5 trading infrastructure for prop firms, brokers, and signal providers.

## Components

| Repo | Description | Language |
| :--- | :--- | :--- |
| [CMB](https://github.com/cipher-suite/CMB) | MT5 Bridge (DLL + EA) | C++ / MQL5 |
| [CNA](https://github.com/cipherbridge/CNA) | Node Agent (Windows service) | Rust |
| [CMG](https://github.com/cipher-suite/CMG) | MT5 Gateway | Rust / axum |
| [CipherTrade](https://github.com/cipher-suite/cipher-trade) | Algorithmic signals | Rust |
| [FX-SIGNAL-COPIER](https://github.com/cipher-suite/fx-signal-copier) | Telegram bot | Python / Rust |
